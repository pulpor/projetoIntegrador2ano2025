## 🎮 Como Usar Agora:

### 1. Instalar Dependências:
```bash
npm run install-all
```

### 2. Desenvolvimento:
```bash
# Backend
npm run dev

# Frontend (em outro terminal)
npm run frontend

# Ou ambos simultaneamente (se tiver concurrently instalado)
npm run dev-full
```

### 3. Produção:
```bash
npm start
```
